## 项目简介 —— vue 分支（高校就业管理系统 前端）

本分支为【高校就业管理系统（CollegeEMS）】的前端部分，采用 Vue 2 框架并结合 Element UI 组件库进行开发，主要负责系统前端界面搭建、用户交互、数据展示及与后端的接口通信。vue 分支实现了前后端分离架构，有效提升了系统的可维护性与扩展性。

### 项目定位

- 适用于高校就业管理相关业务的信息化建设；
- 管理员、学生、企业用户均可通过系统进行便捷操作；
- 专为 Web 端优化，兼容主流浏览器，UI 体验优秀。

### 技术架构

- **Vue 2.x**：主流前端框架，响应式视图、组件化开发。
- **Element UI**：高效美观的企业级 UI 组件库，提升开发效率及界面美感。
- **Vue-Router**：前端路由，支持多页面场景，实现无刷新页面切换。
- **Vuex**：状态管理，保证多组件间状态同步。
- **Axios**：与后端 Spring Boot API 通信，实现数据的异步获取与提交。

### 主要功能模块

1. **用户认证与权限管理**
    - 支持多角色登录（学生、管理员、企业）
    - 权限控制，按角色展示不同模块和操作入口

2. **学生管理**
    - 学生信息录入、修改、批量导入与导出
    - 学生就业信息登记和跟踪

3. **就业信息管理**
    - 招聘信息发布与审核
    - 企业用人信息展示与查询
    - 就业动态、统计分析与报表导出

4. **系统管理**
    - 用户角色分配与管理
    - 数据维护与备份
    - 系统基本设置

5. **数据可视化**
    - 使用图表组件对就业数据、就业率、企业招聘状况等信息进行动态展示

# 高校就业管理系统
> 基于 Spring Boot + Vue2 + ElementUI 开发的前后端分离高校毕业生就业管理平台  
> 适用于高校就业指导中心、二级学院、毕业生、企业四方协同作业

## 项目背景
随着高校毕业生规模逐年扩大，传统基于Excel、人工统计的就业管理模式存在**效率低、易出错、数据不实时、统计难度大**等问题。同时毕业生获取招聘信息渠道分散，企业与学校对接流程繁琐，数据难以统一管理与分析。

为解决上述痛点，本项目采用前后端分离架构，打造一套集**学生就业信息管理、企业招聘管理、岗位发布与申请、就业数据统计分析、系统权限管理**于一体的数字化就业管理系统，实现就业工作全流程线上化、标准化、可视化。

## 技术栈
### 后端技术
- 核心框架：Spring Boot
- 数据库：MySQL 5.7 / 8.0
- 持久层：MyBatis / MyBatis-Plus
- 工具集：Lombok、Hutool、Jackson
- 构建工具：Maven
- 接口规范：RESTful API

### 前端技术
- 框架：Vue2
- UI组件库：ElementUI
- 网络请求：Axios
- 路由：Vue Router
- 构建：Vue CLI / Webpack

### 环境依赖
- JDK 1.8+
- MySQL 5.7+
- Maven 3.6+
- Node.js 14.x

## 核心功能模块
### 1. 系统管理模块
- 用户管理（管理员/教师/学生/企业）
- 角色权限分配
- 菜单管理
- 系统日志、操作日志记录
- 数据字典与基础配置

### 2. 学生管理模块
- 个人信息维护、资料上传
- 就业信息填报（就业单位、岗位、薪资、合同信息等）
- 就业状态跟踪（未就业/求职中/已就业/实习中）
- 岗位浏览、在线投递简历
- 就业协议、证明材料上传

### 3. 教师/就业办管理模块
- 学生就业信息审核
- 批量导入导出学生数据
- 毕业生信息管理、就业情况跟踪
- 就业通知、政策公告发布
- 异常信息处理与数据纠错

### 4. 企业管理模块
- 企业信息注册与认证
- 招聘岗位发布、编辑、下架
- 简历查看、筛选、收藏
- 与学生在线沟通
- 宣讲会信息发布

### 5. 岗位与招聘管理
- 岗位分类、标签、搜索
- 岗位审核机制
- 投递记录管理
- 面试通知推送

### 6. 数据统计与可视化分析
- 各专业就业率统计
- 就业地区分布统计
- 行业分布、薪资水平分析
- 企业需求数量分析
- 图表展示（饼图、柱状图、折线图）
- 支持导出报表（Excel/PDF）

## 系统特色
1. **前后端分离架构**：接口标准化，易于维护与扩展
2. **多角色权限体系**：管理员、教师、学生、企业权限隔离
3. **界面友好易用**：基于ElementUI构建，符合管理系统使用习惯
4. **数据可视化**：多维度统计图表，直观展示就业情况
5. **流程规范化**：就业信息填报—审核—归档全流程线上完成
6. **扩展性强**：支持对接学校统一身份认证、短信通知、文件存储等
7. **部署简单**：内嵌服务器，打包为JAR即可直接运行

## 快速部署指南
### 1. 后端部署
1. 创建MySQL数据库，执行`sql/init.sql`初始化表结构
2. 修改`application.yml`中的数据库连接信息
3. 执行Maven命令：`mvn clean package`
4. 运行启动类或使用`java -jar xxx.jar`启动服务

### 2. 前端部署
1. 进入`frontend`目录
2. 安装依赖：`npm install`
3. 本地启动：`npm run dev`
4. 生产打包：`npm run build`

### 3. 访问地址
- 后端服务：http://localhost:8080
- 前端页面：http://localhost:8081
- 接口文档：http://localhost:8080/doc.html（如集成Knife4j）
